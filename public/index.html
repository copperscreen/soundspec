That's a tool for desiging a POKEY-like chip specs. The chip has a master clock that is divided into (you set how many) derived clocks that drive the counters that switch the output level between zero and a selected volume level. Tune the counters number and width for the notes, LFSR params for the noise and play some music to check if it sounds fine. The only significant feature here that is absent in POKEY is that you can pair channels so that one channel counter is initialized from alternating channels, effectively performing PWM. The volume/frequency envelopes that can be edited for each instrument are processed every vblank. If you have MIDI keyboard (or two!) and your browser supports it you can use it here.

Use Ctrl and Shift to choose an octave for the keyboard. Instruments 1-2 are mapped to midi keyboards / computer keyboard, instruments 4-6 are mapped to Insert, Home and PageUp keys (use them as percussion).
Click an area header to fold the area back. Click the green dot to undo a corresponding instrument edit (if the dot is red there are no undo steps left). To swap instrument settings click a number on the instrument header.
